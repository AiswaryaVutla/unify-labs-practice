Overview

DevConf 2026 is a multi-page conference website built using semantic HTML5. The project focuses on proper document structure, accessible content, and consistent navigation across pages. It was developed as part of Day 1 and Day 2 tasks during my Web Development Internship at Unify Labs.

Files in This Repository
about.html        # Conference overview and details
faq.html          # Frequently Asked Questions
index.html        # Homepage
register.html     # Registration form
schedule.html     # Conference schedule
speakers.html     # Speaker profiles
sponsors.html     # Sponsor information
README.md         # Project documentation

Key Features

Semantic HTML5 structure (header, nav, main, section, article, footer)

Consistent navigation across all pages

Speaker profiles with images and video introductions

Two-day schedule using structured HTML tables

Registration form with multiple input types and validation

FAQ section using definition lists

Clean, readable, and well-indented code

How to Run

Open index.html in any modern web browser to view and navigate the website.
